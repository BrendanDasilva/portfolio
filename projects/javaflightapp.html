JavaFX Flight Reservation App Project

The Flight Reservation App was developed as part of a JavaFX project for our second year Application Development using Java class. It's built around three main parts: Flight, Passenger, and Booking. The Flight class handles all the flight details like where it's going, when, and how much it costs. It makes sure all this information is correct and makes sense, like making sure the flight date is in the future. The Passenger class keeps track of the people flying, including important details like their name and passport number, and checks to make sure this information is properly filled out.

For booking flights, the Booking class connects passengers to their flights, making sure a passenger can't book the same flight more than once. It's smart enough to check if a booking already exists before making a new one. Plus, this app can turn all the flight and passenger details into a simple format (like a CSV file) that's easy to save and look at later. This feature is great for keeping everything organized.

The app uses some neat tricks like regular expressions (a way to check if text is written in a certain format), handling errors smoothly (so it doesn't crash if something goes wrong), and static methods (a way to do something without needing to create a new object first). These features make the app not only easy to use but also very reliable. It's a great example of how to build a practical tool that solves a real problem.
