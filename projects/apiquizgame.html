API Quiz Game Project

API Quiz Game is a dynamic, web-based trivia quiz application designed to challenge users with a wide range of questions across various categories and difficulties. Utilizing the Open Trivia Database API, this interactive game fetches real-time questions to ensure a fresh and engaging experience every time.

Upon starting the game, users can customize their quiz by selecting their preferred category, the number of questions, and the difficulty level. Each question is timed, with a visible countdown to add an element of urgency, encouraging quicker responses. The score decreases as time passes, incentivizing fast and accurate answers to achieve the highest possible score.

The quiz features a sleek, user-friendly interface with a responsive design that caters to desktop and mobile users alike. Progress indicators, current score displays, and high score tracking motivate players to improve with each attempt. Additionally, a Time's up! alert for unanswered questions adds an extra layer of challenge, pushing players to stay alert and think swiftly.

Built with HTML, CSS, and JavaScript, this project demonstrates proficiency in web development fundamentals, API integration, and creating engaging, interactive web applications.
