Simon Says Game Project

The Simon Game project is a web-based adaptation of the classic memory game, designed to test and enhance memory skills through a sequence of lights and sounds. Built using HTML, CSS, and JavaScript, including jQuery for interactive elements, this project demonstrates my ability to create engaging and intuitive user interfaces. The game starts at level zero and progressively increases in difficulty by extending the sequence of lights and sounds the player must memorize and replicate.

Each colored button on the game board corresponds to a specific sound, creating a multi-sensory experience that aids in memorization and pattern recognition. A key aspect of the project was implementing logic to handle user interactions and game mechanics, such as sequence generation, user input validation, and game state management. This project reflects my proficiency in front-end development and my knack for bringing simple concepts to life in a fun and interactive way. Through the development process, I gained valuable experience in problem-solving and enhancing user engagement through visual and auditory feedback.
